# AuraSocials 🚀

Agence Social Media propulsée par l'Intelligence Artificielle

## 🌐 Site en ligne
[aurasocials.vercel.app](https://aurasocials.vercel.app)

## 📋 Description
AuraSocials est une plateforme full-stack combinant un site web professionnel et un CRM social media IA (AuraCRM). Développé dans le cadre du Projet Technique Tutoré — Bachelor 3 Tech For Business 2025/2026.

## ✨ Fonctionnalités
- 🌍 Site web multilingue (50+ langues)
- 🤖 Chatbot IA (API Claude — Anthropic)
- 📊 AuraCRM : gestion clients + réseaux sociaux
- ✦ Générateur IA de publications
- 📅 Calendrier éditorial drag & drop
- 📈 Analytics & KPIs temps réel
- 🔐 Authentification JWT + bcrypt + OAuth2
- 🗄️ Base de données PostgreSQL + Prisma ORM

## 🛠️ Stack technique
| Couche | Technologie |
|--------|------------|
| Frontend | React.js 18 + Vite |
| Styling | Tailwind CSS + CSS custom |
| Routing | React Router v6 |
| Base de données | SQLite (dev) / PostgreSQL (prod) |
| ORM | Prisma v5 |
| IA | Anthropic Claude API |
| Hébergement | Vercel |
| Versionning | Git + GitHub |

## 🚀 Installation locale
```bash
git clone https://github.com/mohamedalisouissi25/aurasocials.git
cd aurasocials
npm install
npx prisma migrate dev
npm run dev
```
